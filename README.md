## Features

- Button & Modal interaction
- Rename nickname (admin only)
- Create text channel
- Ban user with reason
- Permission checking


## Required Bot Permissions

- Manage Nicknames
- Manage Channels
- Ban Members
- Send Messages
- Use Application Commands


## Python Version

Python 3.10+


## ⚠️ Security Notice

Jangan pernah menaruh bot token Discord langsung di dalam kode.
Gunakan environment variable atau file `.env` yang tidak di-commit ke repository.

Jika token pernah ter-publish, segera reset token di Discord Developer Portal.
